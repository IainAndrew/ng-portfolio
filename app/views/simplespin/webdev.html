This project was built with LESS, when I was just getting acquainted with the world of CSS pre-processing.

I first wrote a simple keyframe animation for a 360 degree rotation, then a series of mixins for each animation property - animation-name, animation-duration, animation-direction, animation-iteration-count, and animation-timing-function, complete with vendor prefixes. Finally, I filled out these values for different classes with different functionalities (.spin-1s for a 1 second linear rotation for example). These classes should simply be added to an element in HTML or with Javascript to apply the effect to that element.
