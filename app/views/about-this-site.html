About this site

I created this site not only to showcase my work, but also as an opportunity for me to learn new technologies. I used this opportunity to teach myself AngularJS. I used Angular mostly for its routing capabilites with ui-router, but also its other benefits such as custom directives, data-binding, and my favourite - its modularity.

I also furthered my knowledge of Sass, and made great use of the awesome Bourbon mixin library and Neat grid system.

The design is (very) loosely based on Google's Material Design style guide.

I greatly improved my deployment process with this project, by using the grunt-build-control Grunt plugin to automatically commit and push my built code to my production branch on Github.

Built With:

  • AngularJS
  • Sass
  • Bourbon
  • Neat
  • Grunt
