This project was a lot of fun to put together, as it was my first foray into writing a jQuery plugin.

Essentially, the plugin takes your footer element and gives it fixed positioning, a bottom value of 0 and a z-index value of -100, as well as a nice cross browser compatible box shadow below the previous element to enhance the effect. For the 'reveal', the element before the footer is given a margin-bottom value equal to the height of the footer. This value is recalculated on window resize too, so that the margin value will always be correct even if the footer has a fluid height.

If the footer element happens to be taller than the viewport (on a landscape phone for example), the functionality of the plugin is disabled and the footer remains static, as illustrated below.

I tried to keep the code as DRY as possible and the file as lightweight as possible. The minified version comes in at under 1KB.
