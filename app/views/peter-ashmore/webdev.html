Web Development

One feature I was excited to implement on this site was the animated tape machine. As the user scrolls, the reels of the tape machine rotate. This was achieved with a few lines of jQuery. I used a $(window).scroll() event to trigger CSS transforms on each reel, based on how far the user has scrolled from the top of the window. The function also adds a class to a coloured div behind the tape machine's VU meters, which makes them appear to light up. This class is removed when the user stops scrolling.

Peter wanted the site to have a loading animation, so I decided to use the spinning tape reel as a loader. I implemented this by adding a 'loading' class to body, then removing the class on $(window).load().

For the studio photos, I originally tried a parallax scrolling effect using Skrollr, but the client found it a little distracting, so we settled on simply using background-attachment: fixed.

Peter is an audio engineer, so an audio player was a must to showcase his work. The player is a feed from his Soundcloud account, handled by Toneden.js. This was super easy to implement, but required some styling to fit in with the rest of the page. I removed a lot of the player's features (album cover, follow button, statistics etc) with CSS to leave a nice minimal interface. This player features an awesome animated SVG waveform for which I can't take credit!

As I was using a static host for Peter's website, a server side form submission wasn't an option, so I decided to use Brace.io Forms (now Formspree) to handle the form submission.

For the Instagram feed, I used Instafeed.js, a Javascript plugin which pulls data from the Instagram API via a user ID and oAuth token. This plugin allowed me to write a custom template to display the images in the way that I wanted, including an overlay which appears on hover, displaying the number of likes and comments that photo has.
