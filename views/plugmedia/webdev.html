Web Development

This project was a huge learning experience for me in terms of development. I used a lot of cool development tools (Yeoman, Grunt and Bower), and had a chance to get creative with my code. I had a lot of creative freedom with the website, and I had a few fun ideas for features I wanted to implement.

One of these fun features is a plug that you can click to unplug and 'turn off the lights', leaving the logo glowing like a neon sign. I used jQuery .addClass() to add CSS filters to the text and SVG icon, and change the background overlay to transparent black. I felt that this play on the brand name is an effective way to capture the user's attention.

As you scroll past the header, a large plug descends from the top of the viewport to 'guide' you through the next section. I used Waypoints to detect when the following section is reached, which triggers the plug to animate down into view. I then used Skrollr to animate the 'Process' items in from the side as you scroll. I used the Waypoints 'Sticky' shortcut to 'unfix' the plug element when it reaches the next section, making it appear to 'plug in'. This then triggers a flicker effect on the image, which I created using multiple instances of jQuery .animate() to change the opacity of the overlaying div.

For the 'Showcase' section, I implemented a heavily modified version of this Codrops demo to display the project content. It uses data attributes to display the project with Javascript. In hindsight, I should have built this functionality myself from scratch, as the Codrops demos are not production-ready, and caused various issues with browser compatibility, which needed to be rectified.
